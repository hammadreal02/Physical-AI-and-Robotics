# Quickstart Guide

This guide provides instructions for setting up the development environment and running the Docusaurus project locally.

## Prerequisites

-   Node.js (v18.0 or later)
-   npm or yarn

## Installation

1.  **Clone the repository**:
    ```bash
    git clone https://github.com/your-username/physical-ai-and-robotic.git
    cd physical-ai-and-robotic
    ```

2.  **Install dependencies**:
    ```bash
    npm install
    ```
    or
    ```bash
    yarn install
    ```

## Running the Development Server

1.  **Start the server**:
    ```bash
    npm run start
    ```
    or
    ```bash
    yarn start
    ```

2.  **Open the site**:
    The site will be available at `http://localhost:3000`. Changes to the source files will be automatically reloaded.

## Building the Site

To create a production-ready build of the site, run:

```bash
npm run build
```
or
```bash
yarn build
```

The built files will be located in the `build` directory.
